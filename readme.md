# Тестовое задание avito

## запуск
Для запуска неоходимо наличие установленного docker.
В корне проекта прописать комманду:
```bash
docker-compose up -d
```

## запросы
в приложении реализованы запросы из предоставленной open api документации
после запуска приложения, сервер будет доступен по адресу:
```
localhost:8080
```
